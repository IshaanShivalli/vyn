# Vyn Examples

Run any example from the Vyn REPL with `DOF("examples/<file>.vyn")`.

To run the full non-interactive set:

```vyn
DOF("examples/all_examples.vyn")
```

Files:
- `01_basics.vyn`: variables, arithmetic, strings, lists, dictionaries, ternary expressions, and `typeof`.
- `02_conditionals.vyn`: `IF`, `Elif`, and `Else`.
- `03_loops.vyn`: `forLoop`, `whileLoop`, `repeatUntil`, `forIn`, `break`, and `continue`.
- `04_functions.vyn`: named functions, assigned anonymous functions, defaults, lambdas, and nested returns.
- `05_oop.vyn`: `Class obj Name(...) have`, objects, fields, methods, and `this.field`.
- `06_switch_trycatch.vyn`: `switch`, `case`, `default`, `try`, `throw`, and `catch`.
- `07_libraries_packages.vyn`: standard libraries and package imports.
- `08_reactive_lazy_watch.vyn`: reactive variables, lazy variables, and watchers.
- `09_state_controls.vyn`: locks, snapshots, rollback, ghost variables, and increment/decrement.
- `10_proofs_timed.vyn`: assertions, proof blocks, and timed blocks.
- `11_file_io.vyn`: file I/O helpers.

`IN(...)` input is documented in the README, but it is not included here because these examples are designed to run without waiting for typed input.
