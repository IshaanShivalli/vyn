# Commands

## Install Package

vyn -pack install package

## Install Library

vyn -lib install library

## Delete Package

vyn -del -pack package

## Delete Library

vyn -del -lib library

## Update Everything

vyn -U -A

## Update Libraries

vyn -U -lib

## Update Packages

vyn -U -pack

## Show Package

vyn -show -pack package

## Show Library

vyn -show -lib library

## Search Package

vyn -search -pack package

## Search Library

vyn -search -lib library
